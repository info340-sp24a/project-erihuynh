Project: Topic Proposal

Problem Description

Inspired by other viral personality tests, such as the Cake Resume, Taiwan Expo, and Cosmos tests, we want to create a personality test web application that will allow the user to discover their unique personality traits in a fun and engaging way. Our goal is to provide an immersive experience that not only reveals insights about the userâ€™s personality in an entertaining way but also educates them about different aspects of personality psychology. These persona/personality quizzes serve as an easy and fun way to get to know and connect with others better, whether friends or someone new, and we take these quizzes because they might reveal previously unknown information about ourselves.

While the timeframe is too short to base the personality test on extensive research, we still want to draw inspiration from MBTI. Unlike other applications, our application will allow the user to save their results by logging in, so if they need to reference their personality type, the user will not have to retake the test. Additionally, other applications include the quiz itself, but we want to also include a searchable gallery of the different persona types and their description that are attainable.

User Stories

  • User Story 1: As a social media user, I want to take a personality
    quiz so that I can have fun discovering what personality traits I
    might have and share my results with my friends to see how we are
    similar and different.
  • User Story 2: As a team leader at work, I want to organize a
    personality quiz activity for my team, where we can take the
    quiz together as a bonding experience by fostering better
    understanding of each other and recognizing our potential
    compatibility and differences.
  • User Story 3: As a student in college, I want to take personality
    quizzes to fit in with modern trends and to be able to connect
    better with my peers by having something trendy to talk about
    and share.
  • User Story 4: As a psychology enthusiast, I want to explore a variety
    of personality quizzes online, ranging from serious tests to more
    creative versions to deepen my understanding of personality theories
    while also having fun at the same time.
